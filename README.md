\## Мета лабораторної: Зрозуміти основні концепції систем контролю версій, створювати та керувати Git репозиторіями, фіксувати зміни (commits) та працювати з історією, створювати та об'єднувати гілки (branches), працювати з віддаленими репозиторіями, співпрацювати з іншими розробниками через pull requests, вирішувати конфлікти злиття, використовувати SSH-ключі та токени безпечно, працювати з .gitignore та захищати конфіденційні дані.
